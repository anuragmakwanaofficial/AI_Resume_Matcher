==================================================
  AI Resume Matcher - Quick Start Guide
==================================================

Welcome to the AI Resume Matcher project! Follow these instructions to run the application on your local machine.

---
### 1. Prerequisites
Before getting started, make sure you have the following installed on your machine:
- Python 3.10+
- Node.js (v16+) and npm
- PostgreSQL (running locally on port 5432)

---
### 2. Database Setup
The backend requires a PostgreSQL database to store user and analysis data.
1. Make sure your local PostgreSQL server is running.
2. The default connection expects the username `postgres` and password `2502` on `127.0.0.1`.
3. Open a terminal in the `backend/` directory.
4. Run the database creation script to create the `ai_resume_matcher` database:
   python create_db.py
   (Note: You may need to install `psycopg2` if you haven't already: `pip install psycopg2`)

---
### 3. Backend Setup (FastAPI)
1. Open a terminal and navigate to the `backend/` directory:
   cd backend
2. Create and activate a Python virtual environment:
   python -m venv venv
   # On Windows:
   .\venv\Scripts\activate
   # On Mac/Linux:
   source venv/bin/activate
3. Install the required Python dependencies:
   pip install -r requirements.txt
4. Environment Variables:
   - Copy the `.env.example` file and rename it to `.env`.
   - Update the `.env` file with your actual `GEMINI_API_KEY` and any other specific configurations.
5. Initialize the database schema:
   python init_db.py
6. (Optional) Create an initial admin user:
   python seed_admin.py admin@talentmatch.ai admin123

---
### 4. Frontend Setup (React / Vite)
1. Open a new terminal and navigate to the `frontend/` directory:
   cd frontend
2. Install the necessary Node.js dependencies:
   npm install

---
### 5. Running the Application
If you are on Windows, you can easily start both the backend and frontend at the same time by running the provided batch script from the root directory:
   .\start.cmd

Alternatively, to run them manually:
- Backend: (Inside `backend/` with the virtual environment activated)
  uvicorn app.main:app --reload --port 8000
- Frontend: (Inside `frontend/`)
  npm run dev

Once started, open your web browser and navigate to the URL provided by the frontend server (typically http://localhost:5173).

---
